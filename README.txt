Introduction
Comet crush is a top down shooter game where the player controls a ship trying
to shoot and dodge comets flying towards the player. The goal of the game is to score
the highest score by destroying the most comets.


External Requirements to open:
The user needs python preferably python 3.Also user needs the python library pygame.

Simple step by step guide to run(MACOS)
1. Install Homebrew(enter in terminal): /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
2. brew install python3
3. brew install pygame



Setup and Installation:
Inside command prompt or Terminal for UNIX
run the command: python main.py

How to setup executable:
1. python3 setup.py py2app
Ready to run double-click the folder dist then double-click the file named main



Rules of the game:
1. Shoot or dodge the comets before they touch you

Features:
High Score Tracker
Retry button thats simple
